# kao

```
NAME
        get random kaomoji by keyword

SYNOPSIS
        kao [-all] 顔類

DESCRIPTION
        return a random 顔文字 scoped by a keyword 顔類
        the -all flag prints all 顔文字 for a given 顔類

顔類 KAORUI "FACE TYPE"
        valid 顔類:
          - happy
          - sad
          - angry
          - evs
          - suss
          - hello
          - magic

EXIT CODE
        2  incorrect 顔類
        64 usage error

SOURCES
        kaomoji.ru
```
